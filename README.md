# üè• Insurance Claims & Policy Performance Dashboard (Power BI)

This project is a Power BI dashboard designed to analyze insurance claims and policy performance across multiple dimensions. The dashboard provides a clear overview of claim statuses, premium performance, customer segmentation, and policy activity.  
It is built entirely using **CSV data**, with all data preparation and corrections handled in **Microsoft Excel** (cleaning, standardization, missing value handling), followed by visualization in Power BI.
## üìä **Dashboard Overview**

The report provides actionable insights into:

- Claim trends across policy types  
- Distribution of active vs inactive policies  
- Premium and claim amounts across customer age groups  
- Claim status breakdowns (Rejected, Settled, Pending)  
- Total premium, coverage, and claim amount KPIs  
- Detailed table showing settlement performance by policy type  

These insights help insurance companies monitor performance, identify risk areas, and optimize operational decision-making.
